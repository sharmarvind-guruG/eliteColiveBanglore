# Image Assets

The current site uses temporary remote image URLs in the HTML files.

Before launch, replace them with real property photos in this folder and update the `src` values.

Recommended sizes:

- Hero image: 1600px wide, JPG or WebP, under 350KB where possible.
- Room cards: 900px wide, JPG or WebP, under 180KB each.
- Gallery images: 900-1400px wide, JPG or WebP, compressed for web.
- Logo: SVG or transparent PNG.
